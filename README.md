#### 文件系统

##### 设计要求

* 在内存中开辟一个虚拟磁盘空间作为文件存储器，在其上实现一个多用户多目录的文件系统
* 文件物理结构可采用显式链接（FAT）
* 磁盘空闲空间的管理可选择位示图
* 文件目录结构采用多用户多级目录结构，每个目录项包含文件名、物理地址、长度等信息，还可以通过目录项实现对文件的读和写的保护
* 设计一个较实用的用户界面，方便用户使用
